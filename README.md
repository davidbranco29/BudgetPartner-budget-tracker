# ProfitPilot Budget Tracker

A web-based budget tracking application that allows users to manage income, expenses, and financial reports.

## Features

* Add income and expenses
* Categorize transactions
* Monthly reports
* Tax estimation
* CSV export
* Local storage persistence

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Add transactions using the form
2. Filter and search transactions
3. View reports and summaries
